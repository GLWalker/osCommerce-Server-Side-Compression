Chemo Compress

Bobby Easland aka Chemo
Server Side Compression Script

About

This function was written several years ago by Chemo. It was posted but never released to the masses. It works with any version of osCommerce. Directions are for install on osC 2.3.x series.

  • Compress your osCommerce installation server side
  • Decrease page load time.
  • No need to gzip - in fact, DO NOT USE THE ADMINS GZIP with this script
    Although I have seen no issues using gzip VIA .htaccess
  • Easy to activate/deactivate

Installation:

Upload the new file found in this package to catalog/includes/functions/chemo_compress.php

Open catalog/includes/application top and find: 

  // define general functions used application-wide
  require(DIR_WS_FUNCTIONS . 'general.php');

Directly BEFORE add: 

  // add function for server side compression
  require(DIR_WS_FUNCTIONS . 'chemo_compress.php');
  ob_start('chemoCompress');

Uninstall

Simply undo the above steps, or comment out: 

  ob_start('chemoCompress');

Usage

View your page source. The output should be tightly compressed, otherwise, double check the install steps.
